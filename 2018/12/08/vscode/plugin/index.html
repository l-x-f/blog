vscode 常用插件集锦

2018-12-08 16:12

#vscode #visual studio code #插件

（备忘笔记）

  1. Auto Close Tag：自动闭合 HTML 标签
  2. Auto Rename Tag：修改 HTML 标签时，自动修改匹配的标签
  3. Bracket Pair Colorizer：漂亮的括号着色插件
  4. Chinese (Simplified) Language Pack for Visual Studio Code ：中文语言包
  5. Can I Use：HTML5、CSS3、SVG 的浏览器兼容性检查
  6. Code Runner: 运行选中代码段（支持大量语言，包括 Node）
  7. Color Picker : 拾色器
  8. DotENV: .env 文件高亮，.env 文件 icons 显示
  9. Emoji: 在代码中输入 emoji
  10. ESLint: ESLint 插件，代码检查，高亮提示
  11. Easy LESS：less 文件保存自动生成 css 文件
  12. File Peek：根据路径字符串，快速定位到文件
  13. file-size：当前文件大小
  14. Git History：查看当前文件 git log
  15. HTML CSS Class Completion ：CSS class 提示
  16. HTML CSS Support ：css 提示（支持 vue）
  17. IntelliSense for CSS class names：智能提示 css 的 class 名
  18. JavaScript (ES6) code snippets：ES6 语法代码段
  19. Lodash：Lodash 代码段
  20. Live Server：服务器（相当于使用 nodejs 的 http-server ）
  21. Markdownlint：Markdown 格式提示
  22. minapp：微信小程序标签、属性的智能补全（同时支持原生小程序、mpvue 和 wepy 框架，并提供 snippets）
  23. Npm：运行 npm 命令
  24. Node Exec：node 执行器 f8 直接输出 js 执行结果
  25. Node.js Modules Intellisense ：可以在导入语句中自动完成 JavaScript / TypeScript 模块。
  26. npm Intellisense：导入模块时，提示已安装模块名称
  27. Output Colorizer ：彩色输出信息
  28. open in browser：在浏览器中打开 html
  29. Prettier - Code formatter：代码格式化
  30. Path Autocomplete：路径完成提示
  31. Path Intellisense：另一个路径完成提示
  32. Prettify JSON：格式化 JSON
  33. Regex Previewer：测试正则的插件
  34. SVG Viewer：svg 预览插件
  35. TODO Parser：Todo 管理
  36. TSLint: typescript 检查
  37. vscode-eCSStractor：抽取页面的 class，生成一个 css 文档。
  38. vscode-CSS Peek：class 名字定义跳转。
  39. Version Lens：package.json 文件显示模块当前版本和最新版本
  40. Vetur：Vue 必备语法高亮插件
  41. View Node Package：快速打开选中模块的主页和代码仓库
  42. vscode-icons：文件图标
  43. VueHelper：Vue2 代码段（包括 Vue2 api、vue-router2、vuex2）
  44. Vue 2 Snippets：Vue2 代码段提示
  45. vscode wxml：微信小程序 wxml 语法支持
  46. wxml：微信小程序 wxml 格式化以及高亮组件(高度自定义)
